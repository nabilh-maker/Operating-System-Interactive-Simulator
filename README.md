# 🧠 Operating System Interactive Simulator

An interactive educational simulator for teaching core Operating System concepts.

## 📌 Project Overview

This project demonstrates key Operating System topics in a simple and visual way, including:

* 📄 Paging (Page Replacement Simulation)
* 💾 Memory Allocation (First Fit, Best Fit, Worst Fit)
* 🔒 Banker's Algorithm (Deadlock Avoidance & Safe State)
* 🔁 Swapping (Disk ↔ RAM Visualization)

The simulator is designed for students to learn OS concepts through interactive examples.

This project is deployed using Streamlit.

```bash

```

### 2. Run locally

```bash
streamlit run app.py
```
